# Vinegar

A simple rust-implemented command line tool to codec byte stream.

## Build

``` shell
git clone https://github.com/XTZ206/vinegar.git
cd vinegar
cargo build --release
```

## Usage

``` shell
cargo run --release -- [-d] -i input_file -o output_file
```
